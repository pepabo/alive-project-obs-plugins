## 🔧 インストール方法

![ダウンロード手順イメージ](../../images/install-download.png)

1. 各フィルターのページからLuaファイル（`.lua`）をダウンロード

    - リンクを右クリックして、「リンク先を別名で保存」を選択するとダウンロードできます。

2. OBSメニューの「ツール」→「スクリプト」を選択
3. 「+」ボタンをクリックし、ダウンロードした `.lua` ファイルを選択
4. 「有効なプロパティがありません」と表示されますが、これは正常です

インストールが完了したら、各フィルターのページに記載された手順でフィルターを適用してください。

## 🔧 フィルター適用方法

1. シーンまたはソースを右クリック→「フィルター」を選択

2. 「+」ボタンをクリック→「ドット絵」を選択
